Welcome to the Ampere Developer Portal.

The openR Link In-Vehicle Infotainment (IVI) system, powered by the Android Automotive operating system and Google's automotive services, paves the way for the development of new software for cars, combining the benefits of mobile development with the capabilities of a connected car.

The main interest would be to design a car application using Ampere with Google as an OEM partner. This collaboration would enable the creation of innovative solutions that meet the needs of today's drivers.

Let's explore a new world together!

This portal allows developers to create and test applications on openR Link, using the emulator.

This portal will soon integrate more resources and tools for developers.

Stay tuned!
